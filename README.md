 Pizza Sales Analysis Dashboard
 Project Overview

This project analyzes pizza sales data to understand business performance and identify key trends.
Using SQL for data analysis and Power BI for visualization, the dashboard provides insights into revenue, orders, and customer purchasing patterns.

 Key Performance Indicators (KPIs)

Total Revenue – Total revenue generated from all pizza orders.

Average Order Value – Average amount spent per order.

Total Pizzas Sold – Total quantity of pizzas sold.

Total Orders – Total number of orders placed.

Average Pizzas per Order – Average number of pizzas purchased per order.

📈 Dashboard Visualizations
Hourly Trend for Total Pizzas Sold

A chart showing how pizza sales vary across different hours of the day to identify peak ordering times.

Weekly Trend for Total Orders

A line chart that displays weekly order patterns to identify high-demand periods.

Percentage of Sales by Pizza Category

A pie chart that shows the sales distribution across different pizza categories such as Classic, Supreme, Chicken, and Veggie.

🛠 Tools Used

SQL – Data analysis and querying

Power BI – Data visualization and dashboard creation
